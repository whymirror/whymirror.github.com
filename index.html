Decentralization of _why's Projects

As you may have heard, coding superhero _why the lucky stiff recently deleted all his repos, took down his domains and completely removed his online presence. Nobody knows the reason, and we hope he's doing well.

It's important, however, that his wonderful code live on, and we need your help to do so. We've mirrored what repos we could find and contacted github tech support, who have promised assistance. Whymirror will do what it can to ease the transition but CANNOT be a long-term solution. We need the community (this means YOU) to:

  • Find one or more people to become maintainers of each project

  • Once maintainer(s) have been found, github will 'bless' one as the official repository -- the rest of the network will read "fork of blessedrepo/projectname". (If there are multiple maintainers, the project can set up "camping" as a user -- see rails/rails or starling/starling -- with each maintainer having push rights on the shared repo)

  • The surviving wikis for each project must be moved into github pages within that project. (This is the solution suggested by github's @tekkub) Just:

    • go to http://wiki.github.com/why/camping to find the old, now-uneditable code;
    • Pull from whymirror's branch (git remote add -f -m master whymirror git://github.com/whymirror/PROJECTNAME.git) and checkout the pages branch (git checkout -b gh-pages whymirror/gh-pages)
    • Migrate the content over as best as you can.
    • Submit a pull request to whymirror and to this project's maintainers.

  • Documentation formerly housed on code.whytheluckystiff.net/doc should also be moved to github pages. Our best bet here is probably archive.org.

  • Recover several missing repos -- see the list of repos, below.

  • Finally, if you'd like to help copilot whymirror please get in touch.

Thank you, thank you for whatever contribution you can provide. And to _why: so long and thanks for all the fish^h^h^h^h chunky bacon.

-- Flip Kromer (@mrflip) and Seth Thomas Rasmussen (@greatseth) for the http://github.com/whymirror project
